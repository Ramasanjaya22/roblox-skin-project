# Setup Guide - Roblox Skin Creation

## Prerequisite Tools

### 1. Roblox Studio (Required)
```
Download: https://www.roblox.com/create
Platform: Windows / Mac
Cost: FREE
```

### 2. Blender (Recommended for 3D)
```
Download: https://www.blender.org/download/
Version: 3.x or 4.x LTS
Cost: FREE (open source)
```

### 3. Adobe Illustrator (Optional)
```
Purpose: 2D concept art, texture creation
Alternative: Figma (free), Inkscape (free)
```

---

## Account Setup

### Step 1: Roblox Account
1. Go to roblox.com
2. Sign up with email
3. Verify email
4. Enable 2FA (security)

### Step 2: Creator Hub
1. Go to create.roblox.com
2. Click "Start Creating"
3. Select "Avatar Items" → "Accessories"
4. Complete identity verification (age 13+)

### Step 3: Developer Settings
1. Studio → Settings → Security
2. Enable "API Access"
3. Set up payment (Robux payout method)

## First Skin - Basic Tutorial

### Option 1: Simple Accessory (No Blender)

1. **Open Roblox Studio**
2. **Create new** → "Accessory"
3. **Use parts available:**
   - Mesh Part → customize
   - Decals → apply textures
4. **Test in-game** → play button
5. **Configure item:**
   - Name, description, tags
   - Price in Robux
6. **Upload** → Submit for review

### Option 2: Custom Mesh (Blender)

1. **Blender: Create model**
   - Use R15/R6 rig as reference
   - Low poly (under 10k vertices)
   - Proper UV mapping

2. **Export settings:**
   - Format: FBX
   - Apply scale
   - Include mesh only

3. **Roblox Studio: Import**
   - Import → Mesh
   - Configure collision
   - Add to Accessory

4. **Upload** → Wait for review (1-3 days)

## Common Issues

| Issue | Solution |
|-------|----------|
| Mesh not loading | Check file size (under 5MB) |
| Upload rejected | Check content policy |
| Low sales | Optimize thumbnail, tags |
| Performance issues | Reduce polygon count |
