# Roblox Skin Project

Panduan lengkap untuk mulai membuat dan menjual skin di Roblox.

## Struktur

```
roblox-skin-project/
├── guides/           # Step-by-step tutorials
├── resources/        # Tools dan referensi
├── templates/        # Templates untuk workflow
├── market-analysis/  # Trend dan research
└── docs/            # Dokumentasi tambahan
```

## Quick Start

1. **Setup** → Download tools yang dibutuhkan
2. **Learn** → Pelajari basics Roblox Studio
3. **Create** → Buat first skin Anda
4. **Upload** → Publish ke Marketplace
5. **Monetize** → Track sales dan optimize

---

## Target Goals

| Month | Milestone |
|-------|-----------|
| 1 | Learn basics, create first skin |
| 2 | Upload 2-4 skins, understand marketplace |
| 3 | First sales, iterate based on data |
| 6 | Consistent income 500rb-2jt/bulan |
| 12 | Established catalog, 2-5jt+/bulan |

---

## Income Potential

- **Per skin:** Rp 10.000-50.000 (price you set)
- **Revenue share:** 30% (70% goes to Roblox)
- **Passive:** Same skin can sell repeatedly

**Example:** 10 skins × 100 sales/month × Rp 20.000 × 30% = Rp 600.000/month
